Privacy Statement

The EmergentIR project website is hosted on GitHub Pages. We do not collect personal data directly through forms or registrations, but GitHub may automatically log visitor information such as IP addresses for security and operational purposes, which are considered personal data according to the GDPR. The EmergentIR team does not have access to this data. GitHub processes such data under its own privacy policy, which complies with the GDPR. For more information, please see GitHub’s Privacy Statement.

In all matters concerning the EmergentIR project, we adhere to the GESIS Data Protection guidelines. For more information, please refer to the GESIS Data Protection page.
